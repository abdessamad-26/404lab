Introduction à HTML

  • HTML signifie Hyper Text Markup Language.

  • HTML est le langage de balisage standard pour la création de pages Web.

  • HTML décrit la structure d'une page Web.

  • HTML est composé d'une série d'éléments.

  • Les éléments HTML indiquent au navigateur comment afficher le contenu.

Structure de Base

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titre de la Page</title>
</head>
<body>
    <h1>En-tête Principal</h1>
    <p>Paragraphe de texte.</p>
</body>
</html>

Histoire du HTML

1989-1991 : Naissance du web par Tim Berners-Lee

1995 : HTML 2.0 (premier standard officiel)

1997 : HTML 3.2 et 4.0 (nouvelles balises)

1999 : HTML 4.01 (corrections)

2000s : Tentative de XHTML (peu populaire)

2008-2014 : HTML5 (révolution du web moderne)

Depuis 2014 : HTML en développement continu (living standard)

Éléments HTML

  • <ul> : Liste non ordonnée

    Utilisée pour créer des listes avec des puces.

    Exemple :

    <ul><li>Item 1</li></ul>

  • <ol> : Liste ordonnée

    Utilisée pour créer des listes numérotées.

    Exemple :

    <ol><li>Étape 1</li></ol>

  • <li> : Élément de liste

    Utilisé à l’intérieur de <ul> ou <ol> pour représenter un item.

    Exemple :

    <li>Élément</li>

  • <a> : Lien hypertexte

    Permet de créer un lien vers une autre page ou un site externe.

    Exemple :

    <a href="https://exemple.com">Clique ici</a>

  • <img> : Image

    Affiche une image sur la page. C’est un élément vide (pas de contenu).

    Exemple :

    <img src="image.jpg" alt="Description">

  • <iframe> : Cadre intégré

    Permet d’intégrer une autre page web ou une vidéo YouTube dans la page.

    Exemple :

    <iframe src="https://www.youtube.com/embed/ID"></iframe>

  • <div> : Conteneur générique

    Utilisé pour grouper des éléments, souvent avec du CSS.

    Exemple :

    <div>Contenu ici</div>

  • <section> : Section thématique

    Utilisée pour regrouper des contenus liés (ex: une partie d'article ou un chapitre).

    Exemple :

    <section><h2>Partie 1</h2></section>

  • <nav> : Zone de navigation

    Contient des liens vers d'autres parties du site (menu, barre de navigation).

    Exemple :

    <nav><a href="#home">Accueil</a></nav>

  • <button> : Bouton

    Crée un bouton cliquable, souvent utilisé avec JavaScript ou pour les formulaires.

    Exemple :

    <button>Clique moi</button>
