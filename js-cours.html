Introduction à JavaScript

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives.

Histoire de JavaScript

JavaScript a été créé en 1995 par Brendan Eich.

Évolution de JavaScript

  • 1995 : Création de JavaScript par Brendan Eich chez Netscape.

  • 1997 : Standardisation sous le nom ECMAScript.

  • 2009 : Introduction d'ECMAScript 5 (ES5), ajoutant de nombreuses fonctionnalités.

  • 2015 : Introduction d'ECMAScript 6 (ES6), une mise à jour majeure avec des classes, des modules, et plus.

  • 2020 : ECMAScript 2020, avec des fonctionnalités comme l'opérateur de coalescence nulle et l'opérateur optionnel de chaînage.

Syntaxe de JavaScript

Voici un exemple de code JavaScript :

function greet() {
    alert('Bonjour, le monde !');
}
